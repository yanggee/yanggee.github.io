什么是Hexo？

hexo 是一个快速、简洁、高效的博客框架。hexo使用MarkDown解析文章，在几秒内，即可利用靓丽的主题生成静态网页。hexo出自台湾大学生tommy351之手，是一个基于Node.js的静态博客程序，其编译上百篇文字只需要几秒。

安装

安装Hexo相当简单。由于hexo是基于node.js的静态博客框架，这个教程是针对Mac的，window下面配置的原理是一样，各位看官自行探索。

搜索软件： brew search 软件名, 如 brew search wget
